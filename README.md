# 关于项目
*这是一个在线网页音乐播放器，具有用户模块、音乐模块和播放模块三大主要功能模块*  
*1.用户模块-支持用户管理、登录和注册*  
*2.音乐模块-用于展现歌曲、专辑、歌手信息和特别专栏等*  
*3.播放模块-采用ffmpeg播放音乐*
