# 关于项目
*这是一个在线网页音乐播放器，具有用户模块、音乐模块和播放模块三大主要功能模块*  
*1.用户模块-支持用户管理、登录和注册*  
*2.音乐模块-用于展现歌曲、专辑、歌手信息和特别专栏等*  
*3.播放模块-采用ffmpeg播放音乐*  
**技术栈**
***服务端：SpringBoot/SpringCloud/Redis/MybatisPlus/OpenFeign***
***服务代理：Nginx***
***业务接口注册；Nacos***  
***数据库：MySQL***  
***数据挖掘：Python***
***客户端：Vue3/Pinia/Element-Plus***  
***页内部分文本提供：DeepSeek-R1***  
***歌曲数据来源：QQ音乐/Apple Music***
